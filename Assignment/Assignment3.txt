ANSWER 1
n HTML, tables are used to display data in a structured format on web pages.
 They are helpful for organizing information into rows and columns, making it easier
  for users to read and understand the data presented. 
  
To create a
 table in HTML, you can use the following code as a basic example:

html
<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>


In this code:
- <table> defines the start of the table.
- <tr> defines a row within the table.
- <th> defines a header cell in the table.
- <td> defines a standard cell in the table.
We can customize the number of
 rows and columns by adding more <tr>,
  <th>, or <td> elements as needed.
